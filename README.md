# Neural-Networks-

This repository contains a collection of Neural Network experiments implemented using Python, NumPy, TensorFlow, and Keras. These experiments are designed to demonstrate both fundamental concepts and real-world applications of Artificial Neural Networks (ANN) and Convolutional Neural Networks (CNN).

🚀 Experiments Included
🔹 1. Neural Network using TensorFlow (Low-Level API)
Implemented a simple feedforward neural network
Used TensorFlow (modern equivalent of low-level API)
Demonstrates:
Manual weight initialization
Matrix operations
Basic forward propagation
🔹 2. Neural Network using Basic Python (From Scratch)
Built a neural network using NumPy only
Implemented:
Forward Propagation
Backpropagation
Gradient Descent
Solved the XOR problem using a multi-layer network
🔹 3. Neural Network using Keras
Built a model using Keras Sequential API
Trained on the Iris dataset
Demonstrates:
Dense layers
Activation functions (ReLU, Softmax)
Model training and evaluation
🔹 4. Agriculture Price Prediction using ANN
Implemented a regression model using ANN
Used real-world dataset (California Housing dataset)
Evaluated using:
Mean Squared Error (MSE)
R² Score
